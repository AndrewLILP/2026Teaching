I Haven't Printed In Months

Use this when everything is expected to work normally.
0% Complete

Detailed Walkthrough

Step 1

Open STL

Export STL from Tinkercad and double-click it. Bambu Studio should open automatically.

Why? This is your normal workflow and avoids unnecessary file handling.
Step 2

Check Model Position

Can you see underneath the model?

How flat is flat? The model should appear to be sitting directly on the build plate. If you can see a gap underneath, use Lay Flat.
Step 3

Supports Decision

Will any part of the model print in mid-air?

What are supports? Temporary structures printed underneath overhangs that are removed after printing.
When should I use supports?
  • Roofs
  • Large overhangs
  • Arms sticking out
  • Horizontal features
Step 4

Slice Plate

  • Click Slice Plate
  • Review print time
  • Review filament amount
What does slicing do? Converts your model into printer instructions.
Step 5

Export

  1. Go to the top-right corner.
  2. Click the small ▼ arrow.
  3. Select Export Plate Sliced File.
  4. Save to USB.
Reminder: You found the export option under the ▼ menu, not directly on the main button.
Step 6

Print

  1. Insert USB into P2S.
  2. Open Print menu.
  3. Locate file.
  4. Start print.
Step 7

First Layer Check

Troubleshooting

Printer Cannot See File
  1. Remove USB and reinsert.
  2. Refresh printer file list.
  3. Check file exists on USB using computer.
  4. Confirm Export Plate Sliced File was used.
  5. Export again.
  6. Try a different USB drive.

Cause not yet confirmed. Continue documenting future failures.

Model Is Floating Above Plate Use Lay Flat and ensure the largest flat surface touches the build plate.
Do I Need Supports? If part of the model would print in mid-air, supports are usually required.
Print Won't Stick
  • Clean plate.
  • Verify model is flat.
  • Check first layer.
  • Restart print.
Spaghetti Print Stop the print immediately and check:
  • Adhesion
  • Supports
  • Model orientation
