Grade quick reference — Computing Technology Stage 5

What does each grade look like for this unit?

A Skilfully develops technically concise algorithms. Creativity and innovation in design.
B Develops and tests functional algorithms. Creativity in UI and system design.
C Develops common algorithms and implements them. Sound systems thinking.
D Develops basic algorithms and/or implements them. Basic solutions.
E Follows algorithms and/or partially implements them in a language.
About this chapter

How to use this resource

Each chapter follows one phase of the Design and Production Framework. Work through them in order, or jump to any chapter your teacher assigns. Every chapter ends with notebook prompts — write your answers in your project notebook as evidence of your thinking.

What you will build

  • A Unity simulation of a real-world automated system
  • C# scripts that model sensors, controllers, and actuators
  • A project notebook documenting your design decisions
  • Test cases that validate your system works correctly

Prior knowledge assumed

  • Unity interface and scene navigation
  • Basic C# — variables, if statements, functions
  • GameObjects, components, and the Inspector
  • Colliders and the physics system

Syllabus alignment

This chapter addresses the Software Development: Building Mechatronic and Automated Systems focus area of the NSW Computing Technology 7–10 Syllabus (2022), across all four phases of the Design and Production Framework.
