What you'll build

Core Concepts — What is a Timeline?

Read before you open Unity

Timeline is like a video editor built into Unity. Instead of cutting clips together, you drag keyframes that tell your GameObjects where to be, what rotation to have, and what properties to hold — at every moment in time.
🎬 Playable Director The "projector" — plays the whole sequence on command
🎞️ Track A row in the timeline — one track per object or property
🟦 Clip / Keyframe A block on a track that records a change over time
Playable Director
Component on a GameObject that controls when a Timeline plays
Animation Track
Records position, rotation, scale changes for one object
Keyframe
A snapshot of a value at a specific time; Unity interpolates between them
Interpolation
The automatic in-between frames Unity calculates from your keyframes
Cinemachine
Advanced camera system; not required here but extends Timeline later
Play on Awake
Setting that makes the Timeline run as soon as the scene loads

1

Set Up Your Scene

Create the objects the cutscene will animate

💡If you already have a game scene open, you can skip 1a–1c and use your existing camera and a prop from your scene instead.
  • Create a new scene:
    File › New Scene › Basic (URP)
  • Add a subject for the camera to film. In the Hierarchy right-click and choose:
    3D Object › Cube
    Rename it CutsceneProp.
  • Move the Cube to roughly Position (0, 0, 5) in the Inspector Transform so the camera can see it.
  • Select Main Camera in the Hierarchy. Note its starting position — you'll return to it in Step 3.
⚠️Make sure the camera's Clear Flags is set to Skybox so you get a visible background in the cutscene.
2

Create the Timeline Asset

Open the Timeline window and create a Playable Director

  • Open the Timeline window:
    Window › Sequencing › Timeline
  • In the Hierarchy, create an empty GameObject: right-click and choose:
    Create Empty
    Name it CutsceneDirector.
  • Select CutsceneDirector in the Hierarchy. In the Timeline window you'll see a prompt — click Create.
  • A save dialog appears. Save the Timeline asset as IntroCutscene.playable inside your Assets folder.
Unity has now added a Playable Director component to your CutsceneDirector GameObject, and linked it to the IntroCutscene.playable asset. The Timeline window is now showing that asset — this is where all your tracks will live.
  • In the Inspector on CutsceneDirector, make sure Play On Awake is ticked — this makes the cutscene play automatically when you press Play.
3

Animate the Camera

Create an Animation Track and record two keyframes

  • In the Timeline window, click the + (Add) button on the left side and choose:
    Animation Track
  • A track row appears. Drag Main Camera from the Hierarchy into the track's object slot (the empty field on the left of the track).
  • Click the red record button (●) on the track — the Timeline enters record mode and the Scene View gets a red border.
⚠️While the red record button is active, any Transform changes you make in the Inspector or Scene are automatically saved as keyframes. Don't move the camera outside of recording mode by accident!
  • Make sure the playhead (the white vertical line) is at 0:00. In the Inspector, set Camera Position to your desired start position, e.g. (-3, 2, -1). A keyframe diamond appears on the track at time 0.
  • Drag the playhead to 3:00 (3 seconds). In the Inspector, change Camera Position to your desired end position, e.g. (0, 1, -5). Another keyframe is added.
  • Click the red record button again to exit record mode.
Unity will now interpolate (automatically calculate) all the in-between frames, creating a smooth camera glide from position A to position B over 3 seconds.
💡Test it: press the play button inside the Timeline window (not the main Play button) to preview just the camera move in the Scene View without entering Play mode.
4

Animate the Prop

Add a second track to rotate and scale the CutsceneProp

  • Click + in the Timeline window again and add another Animation Track. Drag CutsceneProp from the Hierarchy into its object slot.
  • Click the red record button on the new track.
  • Move the playhead to 0:15 (15 frames / 0.5 seconds). In the Inspector, set the Prop's Scale to (0, 0, 0) and Rotation to (0, 0, 0). Keyframe saved.
  • Move the playhead to 1:15 (1.5 seconds). Set Scale to (1, 1, 1) and Rotation Y to 180. This creates a spin-in effect.
  • Click the red record button to exit record mode.
Your Timeline now has two tracks running simultaneously. The camera glides over 3 seconds while the prop spins into view half a second after the cutscene starts.
💡Easing curves: right-click any keyframe diamond → Edit Keys to open the Animation Curve editor. Switch from Linear to Ease In/Out for smoother, more cinematic movement.
5

Play & Refine

Test the full cutscene in Play mode and polish

  • Press the main ▶ Play button. The cutscene should begin automatically because Play on Awake is enabled.
  • Watch the Game View. Does the camera move feel too fast? Too slow? Too jerky?
  • Exit Play mode and adjust: drag keyframes left/right on the track to change timing, or right-click keyframes to adjust easing curves.
  • If the camera snaps back to its original position after the cutscene ends, you need to decide what happens next — see the Advanced task below.
✅ Working cutscene checklist: camera starts at position A and glides to position B · prop starts invisible (scale 0) and spins into view · both animations complete before the Timeline ends · no jerky jumps between keyframes.
💡Stop the cutscene with code (optional, Advanced): if you want to trigger game logic after the cutscene finishes, attach this tiny script to your CutsceneDirector GameObject:
using UnityEngine; using UnityEngine.Playables; public class CutsceneEnd : MonoBehaviour { private PlayableDirector director; void Start() { director = GetComponent<PlayableDirector>(); director.stopped += OnCutsceneEnd; // subscribe to event } void OnCutsceneEnd(PlayableDirector pd) { Debug.Log("Cutscene finished — start game logic here"); // e.g. SceneManager.LoadScene("Level1"); } }

Your Tasks

Choose the level that matches where you are

Basic

It moves!

Complete all 5 steps. Your cutscene has a camera glide and a prop that scales in. Press Play and it runs automatically. Take a screenshot of your Timeline window showing two tracks with keyframes.

Intermediate

Polish the feel

Apply Ease In/Out curves to at least two keyframes. Add a third track animating a second prop (e.g. a Sphere that drops from above). Record a short screen capture of your cutscene playing.

Advanced

Trigger the game

Add the CutsceneEnd script. When the cutscene finishes, load a new scene or activate a previously disabled HUD Canvas. Document what event is being used and why events are better than checking state in Update().
