Games Industry

Game Programmer

Writes the code that makes games run — from physics and AI to input handling and save systems. Works closely with designers to translate ideas into working mechanics.

🛠
Skills used
OOP / C# Algorithms Debugging Game engines
📚
HSC Pathway
Software Engineering 11–12
➕
Recommended HSC subjects
Mathematics Advanced Physics English Standard
🎮 What you'd make: Gameplay systems like movement, collision, inventory logic, and save/load features using Unity or Unreal.
Creative Fields

UX / UI Designer

Designs the interfaces, menus, HUDs and overall player experience of a game. Blends visual design with an understanding of how players think and feel.

🛠
Skills used
Design thinking Prototyping User testing Visual hierarchy
📚
HSC Pathway
Software Engineering 11–12 Enterprise Computing 11–12
➕
Recommended HSC subjects
Visual Arts Design & Technology English Advanced
🎨 What you'd make: Wireframes, HUD layouts, menu flows, and accessibility-tested interface prototypes in Figma or Unity UI Toolkit.
Simulation Industry

Simulation Engineer

Builds software that models real-world systems — from flight simulators and surgical training to industrial process modelling and climate prediction.

🛠
Skills used
Systems thinking Mathematics Physics modelling C# / Python
📚
HSC Pathway
Software Engineering 11–12
➕
Recommended HSC subjects
Mathematics Extension 1 Physics Engineering Studies
✈️ What you'd make: Physics-based simulations of real environments — driving, surgical, or emergency response trainers used by professionals.
Creative Fields

Technical Artist

The bridge between art and engineering. Optimises 3D assets, builds shaders and visual effects, and creates tools that help artists work faster inside game engines.

🛠
Skills used
3D modelling Shaders Scripting Blender / Unity
📚
HSC Pathway
Software Engineering 11–12
➕
Recommended HSC subjects
Visual Arts Design & Technology Mathematics Standard
🖼 What you'd make: Particle systems, shader graphs, optimised character rigs, and custom editor tools for artists in a production pipeline.

Game Designer

🛠
Skills used

➕
Recommended HSC subjects
English Advanced Visual Arts Music
📋 What you'd make: Game design documents, balance spreadsheets, paper prototypes and player journey maps for original game concepts.

HSC Pathway
