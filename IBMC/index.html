About this resource

NESA Syllabus Aligned

All content maps to Mathematics K–10 (2022), Technology 7–8 (2023), and Computing Technology 7–10 (2022). Outcome codes appear as pills on every section heading throughout.

IB-Style Inquiry Framing

Every chapter opens with a Key Concept, Related Concepts, Global Context, ATL Skills, and a Statement of Inquiry — framing content as a way of thinking, not just calculating.

Stage-Based, Not Year-Based

Content is organised by stage as NESA intends, giving teachers flexibility across Years 7–8 (Stage 4) and Years 9–10 (Stage 5) without a prescribed year-level split.

Core + Path Structure

Stage 5 chapters cover Core content for all students, with Path (★) extensions clearly marked in violet — building direct foundations for Stage 6 Standard, Advanced, and Extension.

Three Difficulty Levels

Every practice set is graded: F Fluency, PS Problem Solving, R Reasoning — mirroring the Working Mathematically processes from the NESA syllabus.

Self-Contained HTML

Each chapter is one HTML file. No server or build process needed — open in any browser, share via any file transfer, or host on any static web server or LMS.
