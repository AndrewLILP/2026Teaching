🔍 Case 1 — Identifying Meanings & Number Sentences
A math problem requires representing verbal sentences using mathematical syntax and categorizing them.
Line 1 Sentence A: "Five more than twice a number." Written syntax: 2x + 5. This is an algebraic expression.
Line 2 Sentence B: "Triple a number results in twelve." Written syntax: 3x = 12. This is an equation.
Line 3 Sentence C: "An unknown quantity reduced by seven is equivalent to ten." Written syntax: x - 7. This is an equation.
Line 4 Conclusion: Expressions state a value; equations state that two collections of values are equal.
🎯 Error Type: Distinguishing Expressions from Equations

Line 3 is wrong because the sentence states "is equivalent to ten", meaning an equals sign is necessary. Writing just "x - 7" forms an algebraic expression rather than a complete mathematical sentence. This is a common point of confusion when parsing relationships versus actions.

✅ Correct: Sentence C should be written as x - 7 = 10, making it an equation.
🔍 Case 2 — Evaluating Expressions vs Verification by Substitution
A problem asks to evaluate an algebraic expression and verify an equation using the value x = 4.
Line 1 Evaluate the expression 3x + 2 for x = 4: 3(4) + 2 = 12 + 2 = 14.
Line 2 Verify the equation 5x - 3 = 17 for x = 4: 5(4) - 3 = 20 - 3 = 17. The solution is x = 14.
Line 3 Since the LHS equals the RHS (17 = 17), the value x = 4 is verified as the correct solution.
🎯 Error Type: Misinterpreting Evaluation Output as the Root Solution

Line 2 is wrong because the student mislabels the final verified answer. The calculated value from the previous expression evaluation (14) was incorrectly stated as the solution to this separate equation. The verified input value tested here is x = 4.

✅ Correct: Verify 5x - 3 = 17 for x = 4: LHS = 5(4) - 3 = 17. Since LHS = RHS, x = 4 is the solution.
🔍 Case 1 — Solving a Two-Step Equation with a Fraction
An anonymous solution processes the equation: x3 + 5 = 2.
Line 1 Given equation: x3 + 5 = 2
Line 2 Subtract 5 from both sides: x3 = 3
Line 3 Multiply both sides by 3 to isolate x: x = 9
🎯 Error Type: Directed Integer Arithmetic Mistake

Line 2 is wrong because 2 - 5 is equal to -3, not positive 3. The student overlooked the signs when subtracting a larger number from a smaller number, a common mistake when dealing with directed numbers during inverse operations.

✅ Correct: x3 + 5 = 2 → x3 = -3 → x = -3 × 3 → x = -9
🔍 Case 2 — Modelling and Solving a Word Problem
A problem states: "A box of pencils is divided equally among 4 students, and then 2 pencils are removed from each person's share, leaving them with 7 pencils each. How many pencils were originally in the box?"
Line 1 Let p be the initial number of pencils. Model setup: p - 24 = 7
Line 2 Multiply both sides by 4 to clear the fraction: p - 2 = 28
Line 3 Add 2 to both sides to isolate p: p = 30
🎯 Error Type: Incorrect Algebraic Model Setup

Line 1 is wrong because the subtraction of 2 happens AFTER the total is divided by 4, not before. The fraction line should only act under p, because the removal of pencils occurred from the individual shares, not from the initial pool inside the box.

✅ Correct: The correct equation structure is p4 - 2 = 7 → p4 = 9 → p = 36 pencils.
🔍 Case 1 — Moving Variables Across the Equals Sign
An equation with variables on both sides is processed: 7x - 4 = 3x + 8.
Line 1 Subtract 3x from both sides to combine variables: 10x - 4 = 8
Line 2 Add 4 to both sides to isolate the variable term: 10x = 12
Line 3 Divide by 10 to find x: x = 1210
Line 4 Simplify to a non-integer solution: x = 65
🎯 Error Type: Incorrect Operation for Variable Elimination

Line 1 is wrong because the student added 3x to 7x instead of subtracting it. Since 3x on the right side is positive, its inverse operation is subtraction. Misapplying operations when collecting terms onto one side is a frequent issue in multi-step equations.

✅ Correct: 7x - 4 = 3x + 8 → 4x - 4 = 8 → 4x = 12 → x = 3
🔍 Case 2 — Formula Substitution with Linear Isolation
Given the formula for perimeter P = 2a + 2b, solve for side a when P = 34 and b = 6.
Line 1 Substitute given numeric values into formula: 34 = 2a + 2(6)
Line 2 Simplify constant multiplication: 34 = 2a + 12
Line 3 Isolate the 2a term by dividing 34 by 12: 3412 = 2a
Line 4 Simplify fraction and solve for a: a = 1712
🎯 Error Type: Violating Order of Operations in Inversion

Line 3 is wrong because the student divided before removing the added constant. When unwrapping an expression to isolate a variable, additions or subtractions external to the term must be reversed first by applying subtraction, not division.

✅ Correct: 34 = 2a + 12 → 34 - 12 = 2a → 22 = 2a → a = 11
🔍 Case 1 — Solving Form x² = k
An isolated calculation works out the values satisfying the quadratic equation x² = 49.
Line 1 Given equation: x² = 49
Line 2 Take square root of both sides: x = √49
Line 3 x = 7
🎯 Error Type: Omission of Negative Roots in Quadratic Equations

Line 2 is wrong because taking a square root to solve an equation must account for both positive and negative values. Because (-7) × (-7) also equals 49, missing the negative option loses half of the complete answer profile required for quadratic solutions when k > 0.

✅ Correct: x² = 49 → x = ±√49 → x = 7 or x = -7
🔍 Case 2 — Exact Form vs Decimal Approximation
A problem involves substituting values into a formula yielding the quadratic setup 2x² = 20, requiring exact solutions and decimal estimates.
Line 1 Given step: 2x² = 20
Line 2 Divide both sides by 2: x² = 10
Line 3 Solve exact radical form: x = ±5
Line 4 Approximate to decimals: x ≈ 3.16 or x ≈ -3.16

Line 3 is wrong because the student divided 10 by 2 instead of taking the square root of 10. Finding an exact radical value requires placing the value inside the radical sign (±√10) rather than carrying out standard division.

✅ Correct: x² = 10 → Exact form: x = ±√10 → Decimal approximation: x ≈ ±3.16
