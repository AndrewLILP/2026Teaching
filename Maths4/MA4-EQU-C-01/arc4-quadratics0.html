What this arc is about

Quadratic equations of the form ax2 = c are a new kind of equation — one where squaring a number hides the sign, creating two possible solutions that are mirror images of each other about zero.

You'll explore this using an interactive number line, learn to write answers in exact form (e.g., ±√20 = ±2√5) and decimal approximation, and reason about when there are no real solutions at all.

No real solutions — here's why
When c < 0, the equation x2 = c has no real solutions. This is because squaring any real number always gives a non-negative result — you can never get a negative number by squaring. The square root of a negative number is not a real number. We say the equation has no real solutions.
Why two solutions?
Squaring erases the sign: (3)² = 9 and (−3)² = 9. Both positive and negative values produce the same square. When we take the square root to solve x2 = c, we must always write ± to include both mirror solutions. They sit at equal distances from zero on the number line.

When there are no real solutions — and the special case c = 0

Squaring any real number always gives a non-negative result. This means x2 = c has solutions only when c ≥ 0.

If c < 0: no real solutions
x2 = −9 has no real solutions.
There is no real number whose square is negative. √−9 is not a real number.
Special case — c = 0
x2 = 0  →  x = 0
Only one solution — 0 is its own mirror image.

General strategy — Hint Ladder
Look at the right-hand side after any division. Is it positive, zero, or negative? That determines how many solutions exist.
For ax² = c: (1) divide both sides by a → x² = c/a. (2) If c/a < 0: no real solutions. If c/a = 0: x = 0. If c/a > 0: x = ±√(c/a). (3) Simplify the surd if possible by finding the largest perfect-square factor.
Example:   4x² = 24
Divide by 4: x² = 6
Take ±√: x = ±√6
√6 has no perfect-square factors → already simplified
Decimal: x ≈ ±2.449
Verify: 4(6) = 24 ✓
