A — Pronumeral vs Variable

🔡 Pronumeral

A letter that stands in for a number. In an equation like x + 5 = 9, the pronumeral x represents exactly one value — the unknown we solve for.

x + 5 = 9 → x = 4 only

↕️ Variable

When a pronumeral can take many different values, it is called a variable. In 2n + 1, n can be 1, 2, 3, 4… and the expression gives a different result each time.

2n + 1 → n can be any number
💡 Key distinction: Every pronumeral is a pronumeral. It becomes a variable in context when its value is not fixed to a single number. Context matters.

B — Notation Conventions

What we meanLong formConcise algebraic formReason for convention
3 multiplied by y 3 × y 3y The × sign can be confused with the pronumeral x
a divided by 4 a ÷ 4 a4 Fraction notation is clearer and extends to algebra more naturally
a multiplied by itself a × a a² Index notation records repeated multiplication efficiently
1 multiplied by n 1 × n n Writing "1n" is redundant — the coefficient 1 is invisible by convention
b multiplied by −1 −1 × b −b The negative sign carries the −1 coefficient implicitly
b multiplied by a b × a ab Letters placed side-by-side means multiply; conventionally written alphabetically

3 = coefficient of x² x = pronumeral ² = power/index −5 = coefficient of x 7 = constant term 3 terms total

Terms are separated by the + and − operations. Each term is identified by its coefficient and pronumeral part.

✏️ Worked Examples — reveal one step at a time

Example 1 — Notation Conventions

Foundation
Rewrite 5 × m × m ÷ 3 using concise algebraic notation.
Step 1 — Identify the repeated multiplication
m × m = m²
When a pronumeral is multiplied by itself, we use index notation. m × m is m to the power of 2, written m².
Step 2 — Write the coefficient next to the pronumeral (remove ×)
5 × m² = 5m²
By convention, we write the number immediately before the pronumeral without a × sign. The × sign can be confused with the pronumeral x.
Step 3 — Replace ÷ with fraction notation
5m²3
Division is written as a fraction in algebra. The numerator is what is being divided; the denominator is what we divide by.
✅ All steps revealed. This is the concise algebraic form.

Example 2 — Parts of an Expression

Foundation
For the expression 4x − 7, identify: the number of terms, the coefficient of x, the pronumeral, and the constant term.
Step 1 — Count the terms
4x − 7 has 2 terms: the term 4x and the term −7
Terms are the parts of an expression separated by + or − operations. The − sign belongs to the term that follows it, making −7 a negative constant term.
Step 2 — Identify the coefficient
Coefficient of x = 4
The coefficient is the number multiplying the pronumeral. In 4x, the number 4 is being multiplied by x.
Step 3 — Identify the pronumeral and constant
Pronumeral = x  |  Constant term = −7
The pronumeral is the letter (x). The constant term has no pronumeral — its value does not change regardless of what x equals.
✅ All steps revealed.

Example 3 — Translating Words to Algebra

Core
Write an algebraic expression for: "8 more than 3 times a number"
Step 1 — Choose a pronumeral for the unknown number
Let the number = n
We use a pronumeral to represent the unknown quantity. Any letter works, but n (for "number") is a natural choice.
Step 2 — Translate "3 times a number"
"3 times n" = 3n
"Times" means multiply. We write 3n using algebraic convention — coefficient before pronumeral, no × sign.
Step 3 — Add "8 more than" to get the full expression
"8 more than 3n" = 3n + 8
"More than" means add. Notice the order: 3n is written first (the main part), then + 8. "8 more than" ≠ "8 − something".
✅ All steps revealed. The expression is 3n + 8.

Example 4 — Substitution and Evaluation

Core
Evaluate 2a² − 3a + 1 when a = 4
Step 1 — Substitute a = 4 into the expression
2(4)² − 3(4) + 1
Replace every a with 4. Use brackets around the substituted value to keep the structure clear and avoid order-of-operations errors.
Step 2 — Evaluate the power first (order of operations)
2(16) − 3(4) + 1
(4)² = 4 × 4 = 16. Powers are evaluated before multiplication. The 2 is a coefficient — it is not inside the bracket, so it does not get squared.
Step 3 — Perform the multiplications
32 − 12 + 1
2 × 16 = 32 and 3 × 4 = 12. Multiplication before addition and subtraction.
Step 4 — Work left to right with the addition and subtraction
32 − 12 + 1 = 20 + 1 = 21
When only addition and subtraction remain, work left to right: 32 − 12 = 20, then 20 + 1 = 21.
✅ All steps revealed. 2a² − 3a + 1 = 21 when a = 4.

Example 5 — Number Pattern from an Expression

Extension
The expression 3n + 2 generates a number pattern. Write the first five terms and describe what happens to the value each time n increases by 1.
Step 1 — Substitute n = 1, 2, 3, 4, 5 into 3n + 2
n = 1: 3(1) + 2 = 5
n = 2: 3(2) + 2 = 8
n = 3: 3(3) + 2 = 11
Substitute each value of n in turn. The number pattern is built row by row — each row is one substitution.
Step 2 — Complete the remaining substitutions
n = 4: 3(4) + 2 = 14
n = 5: 3(5) + 2 = 17
The pattern is: 5, 8, 11, 14, 17, …
Step 3 — Describe the pattern
Each term increases by 3
The common difference is 3 — this equals the coefficient of n in the expression 3n + 2. For any linear expression an + b, the common difference is always a.
✅ All steps revealed. Pattern: 5, 8, 11, 14, 17 — increasing by 3 each term.

🔍 Error Detective — find the mistake

Each case below shows anonymous working with exactly one error. Read each line carefully, click the line you think is wrong, then click Reveal to check.
🔍 Case 1 — Rewriting repeated multiplication
Task: Rewrite a × a × a using concise algebraic notation.

Click the line that contains the error:

Line 1:   a × a × a multiplies a together 3 times
Line 2:   a is multiplied 3 times, so write 3a
Line 3:   Therefore the answer is 3a
❌ The error is on Line 2 (and Line 3 follows from it)

The working confuses repeated multiplication with repeated addition.

  • 3a means a + a + a (adding a three times)
  • a³ means a × a × a (multiplying a by itself three times)

The exponent records how many times the base appears as a factor in multiplication — it does not count additions.

✅ Correct:   a × a × a = a³   (index/power notation)

Error pattern: confusing coefficient with exponent.

🔍 Case 2 — Evaluating an expression
Task: Evaluate 5 + 2x when x = 3.

Click the line that contains the error:

Line 1:   Substitute x = 3:   5 + 2(3)
Line 2:   Add left to right first:   (5 + 2)(3) = 7 × 3 = 21
Line 3:   Answer = 21
❌ The error is on Line 2

The working reads the expression left to right and adds before multiplying. This violates the order of operations.

  • BIDMAS: Multiplication before Addition
  • 2(3) must be calculated before adding 5
  • 5 + 2 cannot be grouped together — there are no brackets around them
✅ Correct:
5 + 2(3)
= 5 + 6   (multiply first: 2 × 3 = 6)
= 11

Error pattern: left-to-right reading overriding order of operations during substitution.

📖 Word Problems — Newman's Framework

Each problem is broken into 5 Newman's steps: Read → Define → Write → Solve → Answer. Work through each step before viewing the full solution.

Problem 1 — Cards and Cost

1-step
A packet of cards costs $c. Write an expression for the total cost of 5 packets, then evaluate the expression when c = 4.
① READ — What is the problem asking?
The problem asks me to _____________ an expression, then _____________ it when c = _____.
② DEFINE — What does the pronumeral represent?
Let c = the cost of _____________ packet in dollars.
③ WRITE — Build the algebraic expression
Total cost of 5 packets = _____ × c = ___________
④ SOLVE — Substitute and evaluate
When c = 4: total cost = 5 × _____ = $_____
⑤ ANSWER — Write a sentence
5 packets cost $_____ when each packet costs $4.
✅ Full Solution
① Read: Write an expression, then evaluate when c = 4
② Define: Let c = the cost of one packet in dollars
③ Write: Total cost = 5 × c = 5c
④ Solve: When c = 4:   5(4) = $20
⑤ Answer: 5 packets cost $20 when each packet costs $4.

Problem 2 — Layla's Earnings

2-step
Layla earns $m per hour and works h hours. Write an expression for her total earnings. If m = 18 and h = 6, how much does she earn?
① READ — Two unknowns this time. What are they?
The problem involves two pronumerals: m = _____________ and h = _____________.
② DEFINE — What does each pronumeral represent?
m = hourly rate in dollars;   h = number of _____________ worked
③ WRITE — Total earnings = rate × time
Total earnings = m × h = ___________
④ SOLVE — Substitute m = 18 and h = 6
Total earnings = _____ × _____ = $_____
⑤ ANSWER — Write a sentence
Layla earns $_____ if she works _____ hours at $_____ per hour.
✅ Full Solution
① Read: Write an expression for total earnings; evaluate with m = 18, h = 6
② Define: m = hourly rate ($); h = hours worked
③ Write: Total earnings = m × h = mh
④ Solve: mh = 18 × 6 = $108
⑤ Answer: Layla earns $108 if she works 6 hours at $18 per hour.

Problem 3 — Tile Pattern

Multi-step
A tiler lays tiles in rows. The nth row has 4n − 1 tiles. (a) How many tiles are in the 7th row? (b) Which row has 19 tiles?
① READ — Two questions. What does each ask?
(a) asks me to substitute n = _____ into 4n − 1.
(b) asks: which value of n makes 4n − 1 = _____?
② DEFINE
n = row number;   4n − 1 = number of _____________ in that row
③ WRITE — Set up each calculation
(a) Substitute n = 7:   4(___) − 1 = ___
(b) Set expression = 19:   4n − 1 = ___
④ SOLVE
(a) 4(7) − 1 = _____ − 1 = _____
(b) Trial: try n = 5: 4(5) − 1 = _____
⑤ ANSWER
(a) Row 7 has _____ tiles.
(b) Row _____ has 19 tiles.
✅ Full Solution
① Read: (a) forward substitution; (b) reverse — find n from the result
② Define: n = row number; 4n − 1 = tiles in row n
③–④ (a): 4(7) − 1 = 28 − 1 = 27
③–④ (b): 4n − 1 = 19 → try n = 5: 4(5) − 1 = 19 ✓ → row 5
⑤ (a) Row 7 has 27 tiles.   (b) Row 5 has 19 tiles.

Note: Part (b) is an example of working backwards — given the output, find the input. This connects substitution to solving equations, which is explored in later chapters.

✅ Self-Marking Practice

Foundation: Identify, read, and name the parts of expressions; basic substitution.
Q1. Which expression matches: "6 less than a number n"?
"Less than" → subtraction. The key is the order: "6 less than n" → n is the starting amount, 6 is taken away.
Q2. What is the coefficient of x in:   7 − 4x + 2x²?
Find the term that has x (not x²). The coefficient includes any negative sign in front of it.
Q3. Evaluate 3n + 5 when n = 2.
Substitute first: write 3(2) + 5. Then BIDMAS: multiplication before addition.
Q4. Which of these uses correct algebraic notation for b ÷ 5?
The notation convention table in the Concepts tab shows how division is written in algebra. Think: fraction form.
Core: Translate words to algebra, substitute and evaluate, connect expressions to tables.
Q1. Write an expression for: "twice the sum of a number and 4"
"The sum of n and 4" → (n + 4). "Twice that sum" → multiply the whole bracket by 2.
Q2. Evaluate 2p² − p when p = 3.
Write 2(3)² − (3). Evaluate (3)² = 9 first, then 2 × 9 = 18, then 18 − 3.
Q3. Which table of values matches the expression n + 4?
Substitute n = 1, 2, 3 into n + 4 and calculate each result. Then match to the table.
Q4. A rectangle has length 2w and width w. Write an expression for the perimeter, then evaluate when w = 5.
Perimeter = sum of all four sides = 2w + w + 2w + w. Simplify, then substitute w = 5.
Extension: Reverse reasoning, pattern identification, two-variable substitution, and error analysis.
Q1. The nth term of a pattern is 5n − 3. Which term has a value of 42?
Set 5n − 3 = 42. Add 3 to both sides: 5n = 45. What is n?
Q2. What expression generates the number pattern: 3, 7, 11, 15, …?
Find the common difference first (how much it increases each step). This is the coefficient of n. Then use n = 1 to find the constant.
Q3. Evaluate 3(a + b)² when a = 2 and b = −1.
Brackets first: evaluate (a + b) before squaring. Remember adding a negative number: 2 + (−1) = 2 − 1 = 1.
Q4. A student writes: "2x + 3x = 5x²". What error did they make?
Think about what 2x + 3x means: (x + x) + (x + x + x) = how many x's? Does that give x²?
