Core concept

What Makes a Quadratic Different?

Unlike linear equations that have one solution, quadratic equations of the form x² = c have two. Here's why.

Why does x² = 25 have TWO solutions?

Think about what squaring does. Two different numbers both square to give 25:

Positive solution
x = +5
Check: (5)² = 25 ✅
Negative solution
x = −5
Check: (−5)² = 25 ✅
The rule: When you square any non-zero number, the result is always positive. So for any positive value of c, there will always be two values of x that satisfy x² = c — one positive, one negative, equal in size.

Three cases — how many solutions?

c > 0
Two solutions
x² = 9
x = +3 or x = −3
c = 0
One solution
x² = 0
x = 0 only
c < 0
No real solutions
x² = −4
No real number squares to give a negative!

Exact form vs Decimal approximation

Exact Form
x = ±√7
This is perfectly precise — no information is lost. Use when asked for "exact form" or "exact value".
✅ For perfect squares: √49 = 7 (exact integer — no surd needed)
Decimal Approximation
x ≈ ±2.646
Rounded using a calculator. The symbol ≈ shows it's approximate. Use when asked for a decimal answer.
⚠️ Always write ≈ not = when giving a rounded answer.

📈 Reading the graph

Parabola The curve y = x² opens upward. Every y-value above zero is hit twice — once on each side.
Horizontal line y = c cuts the parabola at x = √c (right) and x = −√c (left). Two intersection points = two solutions.

Example 1 — Perfect square

Foundation
Solve x² = 49
→
The equation:
x² = 49
49 is a perfect square (7 × 7 = 49), so the solutions will be whole numbers.
1
Take the square root of both sides
x = ±√49
The inverse of squaring is square rooting. We write ± because both +√49 and −√49 are solutions.
2
Evaluate √49
x = 7   or   x = −7
√49 = 7 exactly (perfect square — no surd needed).
✓
Verify both solutions
(7)² = 49 ✅    (−7)² = 49 ✅

→
The equation:
x² = 11
11 is not a perfect square, so the answer will be a surd.

Take the square root of both sides
x = ±√11
This is the exact form. √11 cannot be simplified — it is a surd.
2

x = √11   or   x = −√11

x ≈ 3.317   or   x ≈ −3.317

Example 3 — Rearrange first (coefficient)

Core
Solve 3x² = 75
→
The equation:
3x² = 75
Before we can square-root, we need x² by itself. There's a coefficient of 3 to deal with first.
1
Divide both sides by 3
x² = 25
3x² ÷ 3 = x². 75 ÷ 3 = 25. Now x² is isolated.
2
Take the square root of both sides
x = ±√25
3
Evaluate
x = 5   or   x = −5
✓
Verify: substitute both values into 3x² = 75
3(5)² = 75 ✅    3(−5)² = 75 ✅

Example 4 — Rearrange first (constant on left)

Core
Solve x² − 18 = 0, giving the exact answer
→
The equation:
x² − 18 = 0
We need to isolate x². Start by adding 18 to both sides.
1
Add 18 to both sides
x² = 18
2
Take the square root of both sides
x = ±√18
3
Simplify the surd (optional but good practice)
√18 = √(9 × 2) = 3√2
Since 18 = 9 × 2 and √9 = 3, we can write √18 = 3√2.
4
Exact answer
x = 3√2   or   x = −3√2
Both forms (x = ±√18 and x = ±3√2) are correct exact answers.

Example 5 — Quadratic from a formula

Core / Extension
A = πr²   Find r when A = 50 (use π ≈ 3.14, answer to 2 d.p.)
→
Write the formula and substitute A = 50, π ≈ 3.14
50 = 3.14 × r²
We substitute the known values. We now have a quadratic in r.
1
Isolate r² — divide both sides by 3.14
r² = 503.14 ≈ 15.924
2
Take the square root of both sides
r = ±√15.924 ≈ ±3.99
Using a calculator: √15.924 ≈ 3.990…
3
Consider context — reject the negative solution
r ≈ 3.99 cm
A radius cannot be negative. In real-world formula problems, always consider whether both solutions make sense in context.

Line 2 correctly writes x = √36, but Line 3 only gives the positive solution. The negative solution x = −6 is missing. Both solutions must be written.

Correct Line 3:   x = ±6   →   x = 6 or x = −6

Common mistake: only writing the positive square root and forgetting ±. This is the most frequent error in this topic.

🔍

Detective Case 2 — Solve 5x² = 80

Line 1: 5x² = 80
Line 2: x² = ±√80 ≈ ±8.94
Line 3: x = ±√8.94 ≈ ±2.99

I think the error is on line:  

🔍 The error is on Line 2.

The correct first step is to divide both sides by 5 to isolate x². Instead, the student square-rooted the entire equation before isolating x². This produced x² = ±√80, which is incorrect — x² can't be negative.

Correct working:
5x² = 80   →   x² = 16   →   x = ±4

Always isolate x² completely before taking the square root.

Formulas leading to quadratics

Formula Problems

Substituting known values into a formula can produce a quadratic equation. Solve for the unknown, and always consider whether both solutions make sense in context.

📋 Strategy: Substitute known values → rearrange to get x² = c → take ±√c → consider context (e.g. can a length be negative?) → state your answer with units.

Q1. Solve x² − 12 = 0 and simplify the surd completely.
💡 Need a hint?
Add 12: x² = 12. To simplify √12, find the largest perfect square factor of 12. 12 = 4 × 3, so √12 = √4 × √3 = 2√3.
Q2. A square has area 50 cm². Using A = s², what is the exact side length?
💡 Need a hint?
s² = 50. Take the positive square root (side lengths are positive). Simplify: √50 = √(25×2) = 5√2.

Step 1: Add 36 → 4x² = 36. Step 2: Divide by 4 → x² = 9. Step 3: x = ±√9.
