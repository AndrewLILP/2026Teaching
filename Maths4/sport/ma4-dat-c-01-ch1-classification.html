Key Definition — Variable
A variable is any characteristic, number or quantity that can be measured or counted. In statistics, it is what we are collecting data about.
Sport examples: a swimmer's time, an AFL team name, a player's batting average, a sprinter's finishing position, a basketball player's height.

The Four Types You Need to Know

Every variable in sport — and in life — can be classified into one of these four types. Each type requires different methods to display and analyse.

🔢
Numerical Discrete
Counted whole numbers. No fractions possible.
Tries scored, wickets taken
📏
Numerical Continuous
Measured values. Any decimal possible.
Race times, heights, distances
🏷
Categorical Nominal
Labels or names. No natural order.
Team name, position, sport
🥇
Categorical Ordinal
Categories with a meaningful order.
Finishing place, performance rating

The Four Variable Types — In Detail

Each type has a specific definition, a key identifying question, and multiple Australian sports examples. Study these before the practice questions.

Numerical
Discrete
Values are counted — they can only take specific, separate whole-number values. You cannot have half a unit.
🔑 Key test: "Can this be a fraction?" If no → discrete.
Australian sports examples
🏉 NRL tries 🏏 Cricket wickets 🦅 AFL goals 🏀 Points scored 🏐 Netball goals 🏊 Laps swum
Numerical
Continuous
Values are measured — they can take any value within a range, including decimals. Precision depends on measurement equipment.
🔑 Key test: "Can this be a non-whole number?" If yes → continuous.
Australian sports examples
🏃 Sprint time (s) 🏊 Freestyle time (s) 🏀 Height (cm) 🏏 Batting average 🦅 GPS distance (km) 🤾 Jump height (m)
Categorical
Nominal
Values are labels or names — they have no natural ordering and no numerical meaning. You cannot say one is "greater than" another.
🔑 Key test: "Is there a natural ranking?" If no → nominal.
Australian sports examples
🦅 AFL team 🏉 NRL club name 🏐 Netball position 🏏 Batting hand 🏊 Stroke type 🏃 Athlete's state
Categorical
Ordinal
Values are ordered categories — there is a clear ranking but the gaps between categories are not necessarily equal.
🔑 Key test: "Are there ordered categories with unequal gaps?" → ordinal.
Australian sports examples
🏃 Finishing place 🏊 Heat→Final 🦅 Player rating 🥇 Bronze/Silver/Gold 🏀 Division (1st/2nd) 🏉 NRL rankings

⚠️ Watch Out — Common Traps

These are the most common classification mistakes students make.

🚨
Jersey numbers are categorical nominal — not numerical. Umpire #7 is not "less than" Umpire #12. The numbers are labels. You can't calculate a meaningful average jersey number.
✅
Batting average is numerical continuous — not discrete. A batting average of 52.4 runs per dismissal is a calculated decimal, not a count of whole runs.
🎯
Finishing positions are ordinal, not discrete — even though we write "1st, 2nd, 3rd", the gaps are unequal (1st and 2nd might be 0.01s apart; 2nd and 3rd might be 2s apart).

🟢 Foundation — Identifying Variable Types
Question 1
An NRL scoreboard records the number of points scored by each team at full time. What type of variable is "points scored"?
Points values: 24–18, 14–6, 38–12, 20–20 (from four recent matches).
Question 2
At the world athletics championships, an athlete's 400m hurdles time is recorded as 52.83 seconds. What type of variable is "race time"?
Times at this standard range from roughly 47 to 55 seconds, measured to the nearest hundredth of a second.
Question 3
A Super Netball club surveys all players and records which team each player belongs to (Swifts, Vixens, Firebirds, Lightning…). What type of variable is "team"?
Question 4
At the school swimming carnival, students finish in positions: 1st, 2nd, 3rd, 4th, 5th. What type of variable is "finishing position"?
1st and 2nd may be separated by 0.04s. 4th and 5th might be 3s apart. The gaps between positions are unequal.
Question 5
A basketball scout rates each player's defensive performance as: Poor / Fair / Good / Very Good / Excellent. What type of variable is this "defensive rating"?

🔴 Extension — Edge Cases & Justification
Question 1
In a 100m sprint final, each athlete is assigned a lane: Lane 1, Lane 2, … Lane 8. What type of variable is "lane number", and why does it matter which type it is?
Question 2
A cricket tournament uses the Duckworth–Lewis–Stern (DLS) method to set revised run targets. A revised target might be "127.4 runs". What type of variable is a "DLS target"?
DLS targets are mathematically derived from complex formulas and can be non-integer values.
Question 3
A researcher classifies Australian football grounds by surface type: Natural Grass, Synthetic Grass, or Hybrid. Another researcher also records average bounce height of the ball on that surface in centimetres. How should these two variables be classified?
Question 4
A sports science study measures blood lactate concentration in NBL basketball players immediately after a sprint test. Results range from 4.2 to 18.7 mmol/L. Meanwhile, researchers also note whether each player experienced cramping: Yes or No. How should these variables be classified?
Question 5
Why is it important to correctly classify a variable before choosing how to display or analyse data? Which answer best justifies this?

Key Vocabulary

Review these terms. Each connects to a distinct concept from the NSW Stage 4 syllabus.

Variable
Any characteristic, number or quantity that can be measured or counted.
e.g. sprint time, team name, medal type
Numerical Discrete
Counted values — only whole numbers are possible.
e.g. tries scored, wickets taken, goals kicked
Numerical Continuous
Measured values — any decimal in a range is possible.
e.g. race time, height, distance thrown
Categorical Nominal
Labels or names with no natural order.
e.g. team name, court position, stroke type
Categorical Ordinal
Categories with a meaningful order, but unequal gaps between them.
e.g. finishing place, performance rating, medal type
Classification
Identifying the type of variable — determines which graphs and statistics are appropriate.
e.g. "batting average is numerical continuous"
