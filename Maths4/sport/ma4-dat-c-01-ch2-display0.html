The Right Graph Tells the Right Story

Eight different graph types are on the NSW Stage 4 curriculum — each one designed for a specific kind of data and a specific purpose. A statistician who picks the wrong graph can accidentally mislead an entire audience. A good one makes a complex dataset instantly clear.

📌 Use a column graph when comparing counts or totals across separate, distinct categories — teams, rounds, players. Bars do not touch.
Source: Hypothetical data for educational purposes | AFL 2024

✅ Required Features Present

  • Title (what, who, when)
  • X-axis label: Quarter
  • Y-axis label: Goals
  • Scale: 0–6 on y-axis
  • Legend for two teams
  • Source listed below

📋 Key Notes

  • Bars do not touch — data is categorical (quarters)
  • Goals = numerical discrete (whole numbers only)
  • Good for: comparing separate groups side-by-side
  • Sydney surged in Q3; Hawthorn dominated Q4
📌 Use a line graph when showing change over time or sequence — the connected line emphasises the trend and direction of change.
Source: Hypothetical data for educational purposes | NSW Swimming 2024

✅ Required Features Present

  • Title (subject, event, season)
  • X-axis label: Race Number
  • Y-axis label: Time (seconds)
  • Scale on both axes
  • Source listed

📋 Key Notes

  • Time = numerical continuous (measured)
  • Downward slope = improving (lower time = faster)
  • Race 5 shows a slight setback — common in real sport
  • Line connects points to show the trend clearly
📌 Use a dot plot when displaying a small-to-medium dataset (up to ~30 values) and you want to show where values cluster, repeat, or stand out as outliers.
Source: Hypothetical data for educational purposes | Cricket Australia 2024

✅ Required Features Present

  • Title (event, venue, year)
  • Labelled number line: Runs Scored
  • Scale: 0–45 with regular tick marks
  • Source listed

📋 Key Notes

  • Each dot = one player's score
  • Stacked dots = repeated values (e.g. 15, 24, 35)
  • Score of 42 is a clear outlier
  • Variable = numerical discrete (whole run counts)
📌 Use a stem-and-leaf plot when you need to display all individual values AND see the distribution shape — ideal for 10–50 values.

📌 Use a frequency histogram when data is numerical and continuous — group values into equal class intervals. Bars MUST touch (no gap). An optional frequency polygon overlays the shape.
Source: Hypothetical data for educational purposes | NSW Athletics 2024

✅ Required Features Present

  • Title (who, what, when, n=)
  • Bars TOUCH — continuous data
  • Class intervals on x-axis
  • Frequency on y-axis with scale
  • Source listed

📋 Key Notes

  • Times = numerical continuous (measured)
  • Peak at 12.0–12.5s (18 students) — modal class
  • Roughly bell-shaped distribution
  • Polygon joins bar midpoints — shows the curve shape
📌 Use a sector (pie) graph when showing how a whole is divided into parts. Each slice is a proportion of 100%. Best when you have 3–6 categories.
Source: Hypothetical data for educational purposes | Suncorp Super Netball 2024

✅ Required Features Present

  • Title (team, competition, year)
  • All slices labelled with category + %
  • Slices add to 100%
  • Legend with colours
  • Source listed

📋 Key Notes

  • Scoring type = categorical nominal
  • Best for: showing parts of a whole
  • Avoid if any slice is very small (<5%)
  • A divided bar graph can show the same thing but is easier to compare across groups
📌 Use a divided bar graph when showing how a total is split into parts — especially useful when comparing proportions across multiple groups.

✅ Required Features Present

  • Title (team, competition, year)
  • Bar spans 0–100%
  • Each section labelled with category + %
  • Percentage scale shown above bar
  • Source listed

📋 Key Notes

  • Shot type = categorical nominal
  • Proportions must add to 100%
  • Advantage over sector: easy to compare multiple bars (e.g. compare two teams)
  • Use colour to distinguish sections clearly
📌 Use a pictogram when displaying simple count data for a general audience. Each symbol represents a fixed quantity — always include a clear key.

Key Vocabulary

Column graph
Bars representing counts for separate categories. Bars do not touch.
e.g. AFL goals per quarter per team
Line graph
Points connected by lines to show change over time or sequence.
e.g. swim times over a season
Dot plot
Individual values shown as dots above a number line. Stacked dots = repeated values.
e.g. individual T20 batting scores
Stem-and-leaf
All original values displayed with stems (tens) and leaves (units). Back-to-back variant compares two groups.
e.g. NRL metres per game
Freq. histogram
Bars for class intervals of continuous data. Bars must touch. Bars equal-width.
e.g. 100m sprint times (50 students)
Freq. polygon
A line connecting the midpoints of histogram bar tops — shows the distribution shape.
Overlay on the sprint time histogram
Sector graph
Circle divided into slices proportional to percentage. All slices = 100%.
e.g. netball scoring breakdown
Divided bar
Horizontal bar split into sections showing proportions. Good for comparing across groups.
e.g. NBL points by shot type
