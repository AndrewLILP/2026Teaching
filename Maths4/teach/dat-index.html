Lesson 1
Classifying Data

Learn what a statistical variable is and how to classify data as numerical or categorical — including the sub-types discrete, continuous, nominal, and ordinal.

  • Define a variable in the context of statistics
  • Classify variables as numerical or categorical
  • Describe numerical variables as discrete or continuous
  • Describe categorical variables as nominal or ordinal
  • Distinguish and compare all four variable types

Lesson 2
Displaying Data

Represent data using histograms, dot plots, stem-and-leaf plots, bar graphs, line graphs, sector graphs and pictograms — and learn how to choose the right graph.

  • Represent datasets using a range of graph types
  • Include sources, titles, labels and scales correctly
  • Select the best graph type for a given dataset
  • Represent a dataset using a statistical infographic

Lesson 3
Interpreting Data

Read, analyse, and draw conclusions from graphs. Identify trends, make predictions, and evaluate whether a graphical representation could mislead the reader.

  • Identify and interpret data displayed on graphs
  • Identify features of graphs to draw conclusions
  • Interpret patterns to make predictions
  • Explain why a graph can lead to misinterpretation
