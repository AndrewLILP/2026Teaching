Variable
In statistics

DefinitionAny characteristic, number or quantity that can be measured or counted in a study
ExamplesHeight, shoe size, eye colour, test score, temperature
Non-examplesThe number 7 on its own (not measured from anything); a person's name
Key ideaA variable VARIES — it takes different values for different people or things
Numerical
Type of variable
Click to flip →
DefinitionA variable whose values are numbers that you can add, subtract, or calculate with meaningfully
ExamplesHeight (cm), number of siblings, test score (%), temperature (°C)
Non-examplesPhone number, postcode — these are number-looking but not numeric variables
Key ideaNumerical = the number itself is meaningful (50 cm is twice 25 cm)
Categorical
Type of variable
Click to flip →
DefinitionA variable whose values are names, labels, or categories — not meaningful numbers
ExamplesFavourite sport, hair colour, school year level, type of pet
Non-examplesNumber of goals scored — that is numerical, not a category
Key ideaCategorical = values are GROUPS or LABELS, not calculated with
Discrete
Numerical sub-type
Click to flip →
DefinitionA numerical variable that can only take specific, separate values — usually whole numbers (you can count them)
ExamplesNumber of students in a class, number of cars in a car park, number of goals
Non-examplesHeight — you can be 171.4 cm, so it is NOT discrete
Key ideaDiscrete = you can COUNT it. You cannot have 2.5 siblings.
Continuous
Numerical sub-type
Click to flip →
DefinitionA numerical variable that can take any value in a range, including decimals and fractions
ExamplesHeight (cm), weight (kg), time (seconds), temperature (°C)
Non-examplesNumber of books owned — you can't own 2.7 books (that is discrete)
Key ideaContinuous = you MEASURE it. Between any two values there are more values.
Nominal
Categorical sub-type
Click to flip →
DefinitionA categorical variable where the categories have NO natural order or ranking
ExamplesEye colour (blue, green, brown), type of pet, favourite sport, blood type
Non-examplesSurvey ratings (Strongly Disagree to Strongly Agree) — those have an order (ordinal)
Key ideaNominal = NAME only. "Blue" is not better or worse than "Brown".
Ordinal
Categorical sub-type
Click to flip →
DefinitionA categorical variable where the categories have a natural order or ranking, but the gaps between them are not equal
ExamplesSurvey ratings (poor/fair/good/excellent), competition places (1st/2nd/3rd), school grades
Non-examplesHair colour — there is no natural order (nominal)
Key ideaOrdinal = ORDER matters. "Good" is ranked above "Fair" but the gap is not measured.

Classification rules summary

Discrete

Can you COUNT it? Values are whole numbers. You cannot be "between" two values.

Continuous

Do you MEASURE it? Any decimal value is possible. Needs a measuring tool.

Nominal

Are the categories just NAMES with no meaningful order? You can't rank them.

Ordinal

Is there a natural RANKING? Categories can be ordered, but the gaps between them are not equal.

Worked Examples

Click "Show Next Step" to reveal each step one at a time. Read the why for each step.

Example 1 Foundation

Question: A student records the number of times each classmate has visited the dentist this year. Is this a numerical or categorical variable? If numerical, is it discrete or continuous?

Step 1 — Ask: Is this a number we can calculate with?
"Number of dentist visits" = 0, 1, 2, 3, ...
These are actual counts we can add or compare. They are meaningful as numbers, so this is a numerical variable.
Step 2 — Ask: Can we COUNT it or MEASURE it?
You count visits: 0, 1, 2, 3 ... (whole numbers only)
You cannot visit the dentist 1.5 times. The values are whole numbers only — so this is discrete.
Step 3 — State the classification
Numerical → Discrete
Always state both levels: the main type (numerical/categorical) and the sub-type (discrete/continuous or nominal/ordinal).
Example 2 Foundation–Core

Question: A scientist records the daily maximum temperature in Sydney for 30 days. Classify this variable fully.

Step 1 — Can the values be calculated with meaningfully?
Temperature values: 21.4°C, 26.8°C, 19.3°C ...
Temperature is measured as a number. We can find averages, differences, etc. So this is numerical.
Step 2 — Count or Measure?
Temperature is measured with a thermometer — can be 21.4, 21.41, 21.412 ...
Any decimal value is possible — you don't "jump" from 21 to 22. This is continuous.
Step 3 — State the classification
Numerical → Continuous
Measured variables like temperature, height, weight, and time are almost always continuous.
Example 3 Core

Question: A survey asks students to name their favourite subject at school. Classify this variable fully.

Step 1 — Are the values numbers we can calculate with?
Responses: "Maths", "English", "PE", "Art" ...
These are words/labels. We can't add "Maths + English". So this is categorical.
Step 2 — Is there a natural ORDER to the categories?
Maths, English, PE, Art — is any subject "ranked above" another?
No — there is no meaningful order to subject names. Maths is not "better" than Art as a category. So this is nominal.
Step 3 — State the classification
Categorical → Nominal
Whenever categories are just names with no ranking, the variable is nominal.
Example 4 Core–Extension

Question: A restaurant survey asks customers to rate their meal as: Poor / Fair / Good / Very Good / Excellent. Classify this variable fully.

Step 1 — Are the values numbers we can calculate with?
Responses: "Poor", "Fair", "Good", "Very Good", "Excellent"
These are labels, not numbers. You can't add "Good + Fair". So this is categorical.
Step 2 — Is there a natural ORDER?
Poor < Fair < Good < Very Good < Excellent
Yes — there is a clear ranking from worst to best. So this is ordinal.
Step 3 — Check: are the gaps between categories equal?
Is the gap from "Poor" to "Fair" the same as "Good" to "Very Good"?
We cannot say. This is what separates ordinal from numerical. The ranking exists but the intervals are not measurable.
Step 4 — State the classification
Categorical → Ordinal
Survey ratings, school grades (A/B/C), and competition places are classic ordinal variables.
Example 5 Extension

Question: A researcher collects the following variables in a study of Year 7 students: (i) postcode of home address, (ii) number of hours of sleep per night, (iii) school attendance rating (Always/Usually/Sometimes/Rarely). Classify each variable.

Step 1 — Classify "postcode"
Postcodes: 2000, 2065, 2088 ...
Although postcodes look like numbers, we cannot meaningfully calculate with them (postcode 2000 + 65 = 2065 is meaningless). Postcodes are LABELS for locations. This is categorical → nominal.
Step 2 — Classify "hours of sleep"
Sleep: 6.5 hrs, 8 hrs, 7.25 hrs ...
Sleep time is measured and can take any decimal value. This is numerical → continuous.
Step 3 — Classify "attendance rating"
Always > Usually > Sometimes > Rarely
These are categories with a natural order (Always is "more" than Rarely), but the gaps are not equal. This is categorical → ordinal.
Step 4 — Summary of all three
Postcode: Categorical → Nominal
Hours of sleep: Numerical → Continuous
Attendance rating: Categorical → Ordinal
Always look past the surface — numbers that are used as codes or labels are categorical, not numerical.

Error Detective

Each case below contains exactly one error in the classification. Find the incorrect step, then reveal the explanation.

Case 1

Scenario: A student classifies "competition place (1st, 2nd, 3rd)" as follows. Find the error.

Step 1: The values are labels (1st, 2nd, 3rd) — not numbers I can add. → Categorical ✓
Step 2: There is no order to these categories. → Nominal ✗
Step 3: Classification: Categorical → Nominal
❌ Error found in Step 2

The student said there is no order — but there IS a clear order: 1st > 2nd > 3rd. This is ordinal, not nominal. Competition places, rankings, and positions always have a natural order.

Correct classification: Categorical → Ordinal

Common error pattern: Confusing nominal and ordinal. The key question is: can these categories be meaningfully ranked? If yes → ordinal.

Case 2

Scenario: A student classifies "the number of cars in a car park each morning" as follows. Find the error.

Step 1: "Number of cars" — I can add, compare and calculate. → Numerical ✓
Step 2: I can measure fractional amounts — e.g. the car park was 2.5 times as full. → Continuous?
Step 3: Classification: Numerical → Continuous ✗
❌ Error in the reasoning and Step 3

The student confused "can be compared as a ratio" with "is continuous". The number of cars must be a whole number — you can't have 2.5 cars. The variable is discrete. Continuous means the variable itself can take any decimal value, not just that it can be used in ratios.

Correct classification: Numerical → Discrete

Common error pattern: Classifying count data as continuous. Always ask: can the variable itself take a decimal value? If not, it's discrete.

Word Problems

Use Newman's five-step framework to work through each problem systematically.

Problem 1 Foundation

A school nurse records the following information about each student: blood type, height (cm), year group, and number of absences this term.

1 — Read

Read the problem carefully. What information is given?

The study records four variables: __________, __________, __________, and __________.
2 — Define

What is the task asking me to do?

I need to classify each variable as __________ or __________, and then as __________.
3 — Write

Write out your reasoning for each variable before classifying.

Blood type is a _____ because...   Height is a _____ because...   Year group is a _____ because...
4 — Solve

Classify each variable fully using the two-level system.

5 — Answer

Write a clear answer in a sentence for each variable.

Full Solution
  • Nominal Blood type (A, B, AB, O) — labels with no natural order. Categorical → Nominal.
  • Continuous Height (cm) — measured, any decimal value possible. Numerical → Continuous.
  • Ordinal Year group (Year 7, 8, 9...) — categories with a natural order. Categorical → Ordinal.
  • Discrete Number of absences — counted in whole numbers only. Numerical → Discrete.
Problem 2 Core

A sports scientist studies marathon runners. She records: finishing time (minutes:seconds), country of origin, number of previous marathons completed, and performance rating (Elite / Advanced / Intermediate / Beginner). Classify each variable and justify your answer.

1 — Read
There are _____ variables recorded in this study.
2 — Define
For each variable I need to determine whether values are __________ or __________, then identify the sub-type.
3–5 — Solve and Answer
Work through each variable using the key questions: "Can I calculate with this number?" and "If categorical, is there an order?"
Full Solution
  • Continuous Finishing time — measured in time, can include decimal seconds. Numerical → Continuous.
  • Nominal Country of origin — names/labels, no natural order. Categorical → Nominal.
  • Discrete Number of previous marathons — counted, whole numbers only. Numerical → Discrete.
  • Ordinal Performance rating — categories with a clear order: Beginner < Intermediate < Advanced < Elite. Categorical → Ordinal.
Problem 3 Extension

A researcher argues that "shoe size" is a discrete numerical variable, while another researcher classifies it as ordinal categorical. Who is correct, and why? Justify your answer using the classification criteria.

1–2 — Read and Define
I need to evaluate _____ positions and decide which classification is more defensible by applying the criteria for __________ and __________.
3 — Write your reasoning
Argument for discrete numerical: Shoe sizes are numbers (4, 5, 5.5, 6...) and we can ___________. However, the gaps between sizes _____________.
4–5 — Evaluate and conclude
The stronger classification is __________ because __________.
Full Solution

Both researchers have a defensible position, but discrete numerical is stronger in most contexts.

For discrete numerical: Shoe sizes are expressed as numbers (4, 5, 5.5), they follow a fixed scale, and a size 8 foot is genuinely larger than a size 6. We can say size 10 is about twice size 5 in a meaningful sense. The values jump in fixed steps (0.5 in Australia/UK), making them discrete.

For ordinal categorical: The argument is that shoe sizes are just codes assigned to foot length ranges — the "number" is a label. In some classification frameworks this is accepted.

This is a genuine edge case in statistics — the important skill is being able to justify your classification with clear criteria, not just naming the category.

Practice

Choose your tier. All questions are self-marking. Use the hints if you're stuck.

Foundation: These questions ask you to identify the variable type from its description. Focus on the key question for each type.
1. A variable whose values are the colours of cars in a car park. Which type best describes this?
Ask yourself: is this a number I can add or calculate with? If not — categorical. Then ask: do the categories have a natural order?
2. Which of the following is a DISCRETE numerical variable?
Discrete = you COUNT it (whole numbers only). Continuous = you MEASURE it (decimals possible).
3. A student says "ordinal and nominal are both categorical, so they are the same thing." What is wrong with this?
Think about competition places: 1st, 2nd, 3rd. These are categories — but they have an order. That is the key difference between ordinal and nominal.
4. Which of these is the best definition of a statistical variable?
The word "variable" contains "vary" — what makes something a variable is that it takes different values for different people or things.
Core: These questions ask you to classify real-world examples. Apply the two-level classification system.
1. A researcher records the postal zones of participants (Zone 1, Zone 2, Zone 3...). How should this be classified?
Can you meaningfully add Zone 1 + Zone 2? No — so it's categorical. Can you order the zones? Yes — so it's ordinal.
2. A doctor records the number of steps each patient walks per day using a fitness tracker. Classify this variable.
Ask: can I add step counts? (Yes → numerical). Can I have a decimal number of steps? (No → discrete).
3. Students are asked "What is your main form of transport to school?" (Walk / Bus / Car / Train / Bicycle). What type of variable is this?
Can you rank the transport modes in a natural order? If not, it's nominal. Is there any number involved? If not, it's categorical.
4. The amount of rain (in mm) recorded at a weather station each day is best described as:
Is rainfall measured? Yes — with a rain gauge. Can it take decimal values? Yes — 3.5 mm is possible. That points to continuous.
Extension: These questions require you to justify your classification and handle edge cases.
1. A student says that "year group" (Year 7, 8, 9, 10) should be classified as discrete numerical because it uses numbers. Which response best evaluates this claim?
The key test: can you meaningfully add Year 8 + Year 9 = Year 17? No. These are ordered labels, not quantities.
2. A researcher records ATAR scores (0–99.95) for university applicants. What is the most precise and justified classification?
Can you meaningfully average ATARs? Yes. Can they take decimal values? Yes (87.45). What does that suggest?
3. A survey uses a 1–5 scale where 1 = Strongly Disagree and 5 = Strongly Agree. A statistician wants to calculate the average score. Which classification supports this choice and why?
To calculate a mean, what must be true about the intervals between values? What classification allows this assumption?
4. Which statement BEST explains the difference between discrete and continuous numerical variables?
Think about number of siblings (0, 1, 2, 3...) vs height (could be 170, 170.1, 170.01...). What is the fundamental difference?
