Frequency

DefinitionHow many times a value or category occurs in a dataset
Examples"Blue eyes" appeared 12 times; the score 7 had a frequency of 5
Non-examplesThe actual data value itself (e.g. "blue") — that's the category, not the frequency
Key ideaFrequency = COUNT of occurrences. Used on the y-axis of many graphs.
Histogram
Graph type
Click to flip →
DefinitionA graph using touching bars (no gaps) to display the frequency of continuous numerical data grouped into intervals
ExamplesHeights of students grouped in 5 cm intervals; test scores in 10-point bands
Non-examplesA column graph for categorical data — those have gaps between bars
Key ideaNO GAPS between bars. Bars touch because the data is continuous.
Dot Plot
Graph type
Click to flip →
DefinitionA graph where each data value is represented by a dot above a number line — good for small datasets
ExamplesNumber of books read by each student; goals scored per game
Non-examplesLarge datasets with many different values — too many dots makes it unreadable
Key ideaOne dot = one data point. Shows the shape of data clearly for small sets.
Stem-and-Leaf
Graph type
Click to flip →
DefinitionA display where each data value is split into a "stem" (tens digit) and a "leaf" (units digit) — preserves the original data
ExamplesTest scores 67, 72, 74 → stems: 6|7, 7|2 4
Non-examplesCategorical data (e.g. colours) — cannot be split into stems and leaves
Key ideaPreserves all original data values. Good for spotting the median and shape.
Sector Graph
Also called pie chart
Click to flip →
DefinitionA circle divided into sectors (slices) where each sector's angle represents a proportion of the whole
ExamplesProportion of students in each sport; budget breakdown by category
Non-examplesData showing change over time — use a line graph instead
Key ideaBest for showing PARTS OF A WHOLE. All sectors must add to 100%.
Infographic
Statistical display

DefinitionA visual representation combining graphs, icons, text and colour to communicate a dataset clearly to a general audience
ExamplesA poster showing school recycling data with icons, percentages, and a bar chart
Non-examplesA raw data table — that is not an infographic

Concepts

Each graph type is suited to different kinds of data. Learn the rules for selecting graphs.

Graph Types at a Glance

Numerical data

Frequency Histogram

For continuous numerical data grouped into intervals. Bars touch because data is continuous.

Numerical data

Dot Plot

Best for small discrete datasets. Each dot represents one data value above a number line.

Numerical data

Stem-and-Leaf Plot

Preserves original data. Good for spotting the middle value and shape of the distribution.

Categorical data

Column Graph

For categorical or discrete data. Bars have gaps between them to show separate categories.

Trend over time

Line Graph

Best for showing how data changes over time. Points connected by lines suggest a continuous trend.

Parts of a whole

Sector Graph

Shows each category as a proportion of the whole. All sectors must add to 100%.

Parts of a whole

Divided Bar Graph

Like a sector graph but in bar form. One bar divided into sections representing proportions.

Pictorial

Pictogram

Uses icons or pictures to represent data. Each icon represents a fixed quantity. Needs a key.

Rules for all graphs — always include:

Title

Clearly describes what the graph shows. Example: "Favourite Sports of Year 7 Students"

Axis Labels

Both axes labelled with the variable name and units where relevant. Example: "Frequency" and "Height (cm)"

Scale

Equal intervals on axes. Scale must start at 0 unless clearly stated otherwise.

Source

Where did the data come from? Example: "Source: 2024 Year 7 survey, Trinity College"

Worked Examples

Click "Show Next Step" to reveal each step. Each example covers a different graph type.

Example 1Frequency Histogram

Question: The heights (in cm) of 20 students were recorded. Construct a frequency histogram using the data grouped into intervals: 150–155, 155–160, 160–165, 165–170, 170–175. Frequencies: 2, 5, 8, 4, 1.

Step 1 — Identify the graph type
Height is continuous numerical data grouped into intervals. A frequency histogram is appropriate — bars will touch because the intervals are continuous.
Step 2 — Set up the axes
Horizontal axis (x): Height in cm, labelled with interval boundaries (150, 155, 160, 165, 170, 175). Vertical axis (y): Frequency, starting at 0, scale up to at least 8 in equal steps.
Step 3 — Draw the bars
Notice bars touch — no gaps. This signals continuous data. Each bar spans exactly one interval.
Step 4 — Add required elements
Title: "Heights of 20 Students". Axis labels: "Height (cm)" and "Frequency". Scale: 0–8 in equal steps. Source: note where the data came from. These four elements must always be present.
Example 2Stem-and-Leaf Plot

Question: Test scores for 12 students: 63, 67, 71, 74, 74, 78, 80, 81, 85, 88, 92, 95. Construct a stem-and-leaf plot.

Step 1 — Identify stems
The tens digits are: 6, 7, 8, 9. These become the stems. List them in order from smallest to largest.
Step 2 — Write each leaf
Each leaf is the units digit. Write leaves in order from smallest to largest in each row. Include a key!
Step 3 — Read back the data
Check: Row 6 → 63, 67. Row 7 → 71, 74, 74, 78. Row 8 → 80, 81, 85, 88. Row 9 → 92, 95. All 12 original values are preserved — this is a key advantage of stem-and-leaf plots.
Example 3Dot Plot

Question: The number of goals scored by a team in 15 games: 0, 1, 0, 2, 3, 1, 1, 2, 0, 4, 2, 1, 3, 2, 1. Construct a dot plot.

Step 1 — Draw a number line
The values range from 0 to 4. Draw a horizontal number line from 0 to 4 with equal spacing. Label it "Goals Scored".
Step 2 — Place one dot per value
Stack dots vertically above each value. Tally: 0 appears 3 times, 1 appears 5 times, 2 appears 4 times, 3 appears 2 times, 4 appears 1 time. Total = 15 ✓
Step 3 — Read the display
The tallest stack (most dots) is at 1 — the most common number of goals. The shape shows most games had 1–2 goals. This is easy to see on a dot plot for small datasets.
Example 4Selecting graph type

Question: A student has collected data on the favourite subjects of 80 Year 7 students. Which graph type should they use and why?

Step 1 — Identify the type of data
"Favourite subject" is categorical data — the values are labels (Maths, English, PE, etc.), not numbers. This rules out graphs designed for numerical data (histogram, stem-and-leaf, dot plot).
Step 2 — Identify the purpose
We want to compare how many students chose each subject. This is a frequency comparison across categories — not showing proportions or trends over time.
Step 3 — Select and justify
Best choice: Column graph (or bar graph). Columns show frequency for each category. Gaps between bars correctly signal that the categories are separate, not continuous. A sector graph could also work if showing proportions is the goal.
Example 5Line Graph

Question: A school records average monthly rainfall (mm) over 6 months: Jan 80, Feb 65, Mar 110, Apr 95, May 55, Jun 40. Display this data appropriately.

Step 1 — Identify data type and purpose
Monthly rainfall over time — this is continuous numerical data showing change over time. A line graph is most appropriate: it shows the trend across time clearly.
Step 2 — Draw the line graph
Dots mark exact values; the line shows the trend between months. March had the highest rainfall, June the lowest.

Error Detective

Each case contains exactly one error. Identify it, then reveal the explanation.

Case 1Graph construction

Scenario: A student draws a histogram to display the number of pets owned by students in a class (values: 0, 1, 2, 3, 4). They draw bars with gaps between them. Evaluate their approach.

Step 1: I have numerical data (number of pets) — a histogram is an appropriate graph type. ✓
Step 2: I will use the frequency of each value on the y-axis and the number of pets on the x-axis. ✓
Step 3: I drew gaps between the bars so the reader knows these are separate values. ✗
❌ Error in Step 3

In a histogram, bars must NOT have gaps. But wait — there is a deeper issue here. "Number of pets" is discrete data (0, 1, 2, 3...), not continuous. A histogram with touching bars is designed for continuous grouped data. For small discrete datasets, a dot plot or column graph (with gaps) is more appropriate.

Better choice: A dot plot (for a small class dataset) or a column graph with gaps (to signal discrete separate values).

Common error pattern: Using a histogram for any numerical data. Histograms are specifically for continuous data grouped into intervals.

Case 2Missing graph elements

Scenario: A student produces the following column graph for favourite sports data. Identify what is missing.

❌ Four required elements are missing
  • Title — there is no heading telling the reader what this graph shows
  • Y-axis label — the vertical axis has no label (it should say "Frequency" or "Number of Students")
  • Y-axis scale — there are no numbers on the vertical axis, so values cannot be read
  • Source — no information about where the data came from

Common error pattern: Only labelling the x-axis categories and forgetting the y-axis, title, and source. All four must be present in every graph.

Word Problems

Use Newman's five-step framework to work through each problem.

Problem 1Foundation

A class of 24 students was asked how many books they read in the last month. The results were: 0, 1, 0, 2, 1, 3, 2, 0, 1, 1, 2, 4, 1, 3, 0, 2, 1, 1, 2, 0, 3, 1, 2, 1. Display this data in an appropriate graph and justify your choice.

1 — Read
The dataset contains _____ values. The variable is "number of books read", which is a __________ variable.
2 — Define
I need to select a graph type for __________ data with _____ possible values.
3 — Write
A __________ graph is appropriate because __________. I would NOT use a __________ because __________.
4–5 — Solve and Answer
Construct the graph. Include: title, axis labels, scale, source. Write a sentence justifying your graph choice.
Full Solution

Graph choice: Dot plot. The data is discrete numerical with a small range (0–4). A dot plot shows every data value clearly and the shape of the distribution is easy to see with a small dataset of 24 values.

NOT a histogram: histograms are for continuous data in intervals. NOT a stem-and-leaf: with only 5 possible values, a stem-and-leaf would only have one stem row per value (not useful here).

Problem 2Core

A school canteen records daily sales for five food items over one week. Which graph would best compare total sales for each item, and which would best show how daily sales change over the week? Justify both choices.

1–2 — Read and Define
This question asks for _____ different graphs because there are _____ different purposes.
3–5 — Solve and Answer
For comparing totals: __________ because __________. For showing change over time: __________ because __________.
Full Solution

Comparing totals across items: Column graph. Each food item is a separate category. A column for each item with height showing total sales makes comparison easy.

Showing daily change over the week: Line graph (one line per item, or one line for total sales). A line graph shows how a value changes over time — the slope of the line makes trends visible.

This illustrates that the same dataset can be displayed in different ways depending on the question being asked. Graph choice must always be justified by purpose.

Problem 3Extension

A student wants to create a statistical infographic about their school's environmental sustainability. They have: (a) percentage of waste recycled each month for a year, (b) breakdown of waste types (paper, plastic, food, other), (c) number of trees planted each year for the past 5 years. Design the infographic structure, justifying the graph chosen for each dataset.

1–3 — Analyse each dataset
Dataset (a) shows __________ over __________. Dataset (b) shows __________. Dataset (c) shows __________.
4–5 — Design and justify
For (a): __________ because __________. For (b): __________ because __________. For (c): __________ because __________.
Full Solution
  • (a) Monthly recycling %: Line graph — shows how the recycling rate changes over 12 months. Trends (improving/declining) are clearly visible.
  • (b) Waste types breakdown: Sector graph — each waste type is a category, and we want to show each as a proportion of total waste. All sectors sum to 100%.
  • (c) Trees planted each year: Column graph or dot plot — discrete data (whole trees) over 5 separate years. Columns show comparison across years clearly.

A well-designed infographic would arrange these three graphs with a clear title, colour scheme, icons to reinforce the environmental theme, and source information.

Foundation: Match graph types to their definitions and key features.
1. Which graph type uses bars that TOUCH each other with no gaps?
Think about what "continuous" means — values have no gaps between them. How should the bars reflect that?
2. A stem-and-leaf plot shows data: 4 | 2 5 8. What data values does this represent?
Remember: Stem = tens digit (before |). Leaf = units digit (after |). Combine them: stem then leaf.
3. Which of the following MUST be included in every graph?
What would a reader need to understand the graph without any extra explanation?
4. What does "one dot = one data point" describe?
Which graph type stacks individual dots above a number line?
Core: Select the most appropriate graph type for each scenario.
1. You want to display the proportion of students choosing each lunch option. Which graph is BEST?
The key word is "proportion" — which graph shows parts of a whole?
2. A dataset contains 200 heights (in cm). Which graph is MOST appropriate?
Height is continuous. 200 values is a large dataset. Which graph handles large continuous datasets best?
3. A researcher records the average global temperature each year from 2000 to 2024. Which graph best shows whether temperatures are rising?
The purpose is to see the TREND over time. Which graph type makes trends most visible?
4. A student draws a column graph for the data "heights of 30 students in cm". What is the main error?
Is height categorical or numerical? If numerical, is it discrete or continuous? Which graph type matches?
Extension: Evaluate graph choices and justify decisions.
1. A student argues: "A sector graph is always better than a column graph for categorical data because it looks more professional." Evaluate this claim.
When is a sector graph most useful? When is a column graph most useful? Compare the purposes.
2. A frequency polygon connects the midpoints of histogram bars with a line. What is the advantage of a frequency polygon over a histogram?
Think about what happens when you try to put two histograms on the same graph. What problem arises? How does a line solve it?
3. A pictogram uses 🍆 icons where each icon represents 10 students. There are 3.5 icons for "Swimming". What issue does this cause?
What frequency does 3.5 icons represent if 1 icon = 10? Is there a cleaner way to represent this?
4. A researcher has monthly sales data for three products over 12 months. Which combination of graphs is most useful for a complete analysis?
There are two questions to answer: (1) how do sales trend over time? (2) which product sold most overall? Different graphs suit different questions.
