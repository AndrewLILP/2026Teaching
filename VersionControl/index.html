Lesson 1 – Setup & First Publish

Learning Goals

  • Create a GitHub account
  • Install VS Code
  • Install GitHub Desktop
  • Create a repository
  • Publish first webpage

Steps

  1. Sign up to GitHub
  2. Create a repository named: username.github.io
  3. Clone repository using GitHub Desktop
  4. Create an index.html file
  5. Commit and push changes
  6. Visit https://username.github.io

Key Vocabulary

  • Repository
  • Commit
  • Push
  • Deploy
