From one square to the whole grid

To check whether a value is allowed in a square, you need to know what's already in that square's row, column and 3×3 box. None of those are single values — each one is up to nine cells you need to look at. Checking nine cells one at a time, by writing nine separate lines, would work for exactly one square. You need something that scales to all 81.

Branching, revisited

Selection — branching — is what lets a loop do something useful with what it finds, rather than just visiting every cell for no reason.

Branching inside a scan As the loop visits each cell, it asks: is this cell empty, or does it already hold a value? Only the cells that already hold a value need to be added to the "already used" list — branching is what makes that distinction.

You can also combine conditions with AND / OR — exactly what the Lesson 6 pseudocode did when it checked a value against the row, column and box all at once.

Scanning the grid

Finding the next empty square means visiting every cell in order — a nested loop, outer loop over rows, inner loop over columns:

Pseudocode — find the next empty square

FOR row = 0 TO 8
    FOR col = 0 TO 8
        IF grid[row][col] == 0 THEN
            RETURN row, col
        ENDIF
    ENDFOR
ENDFOR
RETURN "no empty square found"

Let's build the box scan together — the trickiest of the three

Rows and columns are straightforward to scan because the index runs cleanly from 0–8. A 3×3 box is trickier, because you first need to work out which box a cell belongs to, using integer division:

Pseudocode — scan a cell's 3×3 box

box_row_start = (row DIV 3) * 3
box_col_start = (col DIV 3) * 3
box_values = []

FOR r = box_row_start TO box_row_start + 2
    FOR c = box_col_start TO box_col_start + 2
        IF grid[r][c] != 0 THEN
            box_values = box_values + [grid[r][c]]
        ENDIF
    ENDFOR
ENDFOR

For example, the cell at row 4, column 7: box_row_start = (4 DIV 3) × 3 = 3, and box_col_start = (7 DIV 3) × 3 = 6 — so the box scan checks rows 3–5 and columns 6–8.
