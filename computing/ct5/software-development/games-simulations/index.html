How to use this sequence

1 Start with the Theory page if you haven't — it covers the game loop, logic building blocks, design lenses, and ethics before any code.
2 Work through Pages 1, 2, and 3 in order. Each page builds on the previous — the portal in Page 2 uses the player from Page 1; the game state in Page 3 uses the trigger pattern from Page 2.
3 Pages 4 and 5 are capstone projects. There are no step-by-step walkthroughs — you receive a brief, pseudocode, and a sprint plan, and build the rest yourself. Write pseudocode before opening Visual Studio.
4 Record your progress in your project notebook throughout — sprint logs, bug logs, test tables, and reflections. These are your evidence for the D&P framework.
