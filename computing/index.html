Framework

Design & Production phases

Phase 1

Identifying & Defining

Clarify the problem, establish requirements, and define success criteria.

Phase 2

Researching & Planning

Investigate existing solutions, generate ideas, and plan the approach with pseudocode and design.

Phase 3

Producing & Implementing

Build the solution iteratively using sprints, following pseudocode before code.

Phase 4

Testing & Evaluating

Test against criteria, reflect on outcomes, and evaluate using evidence from the project notebook.

CT5 · Stage 5

Course outcomes

CT5-OPL-01

Designs, produces and evaluates algorithms and implements them in a general-purpose programming language

CT5-DES-01

Designs and creates user interfaces and the user experience

CT5-DAT-01

Explains how data is stored, transmitted and secured in digital systems

CT5-DPM-01

Applies iterative processes to define problems and plan, design, develop and evaluate computing solutions

CT5-SAF-01

Selects and applies safe, secure and responsible practices in the ethical use of data and computing technology

CT5-EVL-01

Understands how innovation, enterprise and automation have inspired the evolution of computing technology

CT5-COL-01

Manages, documents and explains individual and collaborative work practices

CT5-COM-01

Communicates ideas, processes and solutions using appropriate media

Stage 5 · Year 10

Course Performance Descriptors

Grade Understanding Problem solving Programming Design & UX
A
Extensive understanding of enterprise, innovation & automation on computing evolution Skilfully applies iterative processes; highly effective solutions using computational, design & systems thinking Skilfully develops, tests & implements technically concise algorithms; selects safe, secure & ethical data practices Demonstrates creativity & innovation in UI/UX design; selects relevant data, media & processes across contexts
B
Thorough understanding of enterprise, innovation & automation Applies iterative processes; develops effective solutions using all three thinking types Develops, tests & implements functional algorithms; selects & applies safe, secure & ethical data practices Demonstrates creativity in UI/UX; selects relevant data, media & processes to communicate appropriately
C
Sound understanding of enterprise, innovation & automation Applies iterative processes; develops sound solutions using computational or design or systems thinking Develops common algorithms and implements them in a general-purpose language; applies safe, secure & ethical data practices Designs & implements user interfaces; selects appropriate data, media & processes
D
Basic understanding of enterprise & innovation on computing evolution Uses processes to produce solutions; develops basic solutions using one thinking type Develops basic algorithms and/or implements them; uses data safely and responsibly Implements basic elements of UI design; uses data to communicate basic information
E
Identifies evolution of and/or innovations in computing technology Identifies processes that may produce a computing solution Follows algorithms and/or partially implements them; works safely with data Identifies UI elements that contribute to user experience; uses data to communicate information in a very limited way
