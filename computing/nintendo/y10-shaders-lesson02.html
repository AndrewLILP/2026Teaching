Texture Sampling & Emission

Lesson 1 left one part of the collectible orb's UnlitGlow material unexplained on purpose: the actual glow. That glow isn't Base Color at all — it's a separate Fragment input called Emission, paired with a texture instead of a flat colour. This lesson finishes that deconstruction, then has you build both into your own shader.

Producing & Implementing

Lesson 2 Brief

Studio context: "A flat colour was a good start, but nothing in this studio ships with placeholder materials. I want a real texture driving Base Color, and a genuine glow — not a bright colour that only looks like it's glowing. If Emission doesn't actually bloom, it isn't finished."

You'll need your working Lesson 1 shader graph as a starting point, plus any small test texture (a simple pattern works fine) and access to your scene's URP Volume settings.

Identifying & Defining

Finish Deconstructing UnlitGlow

Reopen UnlitGlow from the collectible orb. This time, trace what feeds into Emission specifically — a separate Fragment input from Base Color entirely. You should find an HDR Color node, likely with an intensity value well above 1.

Why "well above 1"?

A normal colour value tops out at 1 (fully bright). HDR (High Dynamic Range) colours can go higher — 2, 5, even 10 — specifically so a post-processing effect called Bloom can detect "this is genuinely bright" and add a soft glow around it. A colour value of exactly 1 might look bright on screen, but won't trigger Bloom at all.

Producing & Implementing

Build Texture & Emission

Extend your Lesson 1 graph. Predict your node plan in pseudocode first, then compare against the annotated graph recipe below.

1 Sample a Texture into Base Color Produce

Replace your flat Color node with a texture: add a Texture2D node (exposed, so you can swap images per-material), a UV node, and a Sample Texture 2D node connecting them together, feeding into Base Color.

Pseudocode 
BEGIN BuildTexturedColour()
  CREATE Texture2D node (myTexture), EXPOSED = true
  CREATE UV node
  CREATE SampleTexture2D node
  CONNECT myTexture, UV TO SampleTexture2D
  CONNECT SampleTexture2D.RGBA TO Fragment.BaseColor
END
ShaderGraph Nodes 
Texture2D (exposed) ──▶ Sample Texture 2D.Texture
UV            ──▶ Sample Texture 2D.UV
Sample Texture 2D.RGBA ──▶ Fragment.Base Color
2 Add Tiling & Offset Produce

Insert a Tiling And Offset node between your UV node and the Sample Texture 2D node's UV input, so the texture can repeat or shift without touching the model's actual geometry.

Pseudocode 
BEGIN AddTilingOffset()
  CREATE TilingAndOffset node (tiling, offset)
  CONNECT UV TO TilingAndOffset.UV
  CONNECT TilingAndOffset.Out TO SampleTexture2D.UV
END
ShaderGraph Nodes 
UV ──▶ Tiling And Offset.UV
Tiling And Offset.Out ──▶ Sample Texture 2D.UV
3 Emission with a Genuine HDR Colour Produce

Add an HDR Color node, set its intensity above 1 in the colour picker, and connect it to Fragment's Emission input — a completely separate input from Base Color.

Pseudocode 
BEGIN BuildEmission()
  CREATE HDRColor node (glowColour, intensity > 1)
  CONNECT glowColour TO Fragment.Emission
END
ShaderGraph Nodes 
HDR Color (intensity: 2.5)
  └─ Out ──▶ Fragment.Emission
4 Enable Bloom in the URP Volume Produce

Open your scene's Global Volume, add a Bloom override if one isn't already present, and enable it. Without this step, your HDR Emission is technically correct but produces no visible glow at all.
