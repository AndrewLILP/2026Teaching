Key Vocabulary

Click a card to flip it and reveal the definition.

Linear relationship
Definition: A pattern that changes by the same constant amount each step, so it can be described by a rule in the form y = (coefficient)x + (constant).
Example: 2, 5, 8, 11 — increases by 3 each time, rule y = 3x − 1.
Non-example: 1, 4, 9, 16 — the gaps keep growing, so this isn't linear.
Coefficient
Definition: The number multiplied by x in a rule — it tells you how much y changes each time x increases by 1.
Example: In y = 3x + 1, the coefficient is 3.
Non-example: The "+1" is not the coefficient — that's the constant.
Constant
Definition: The fixed number in a rule — the value of y when x = 0.
Example: In y = 3x + 1, the constant is 1.
Non-example: The "3" is not the constant — that's the coefficient.
Non-linear
Definition: A pattern that does NOT change by a constant amount each step — its graph is curved, not straight.
Example: 1, 4, 9, 16 (square numbers).
Non-example: 2, 5, 8, 11 — constant difference of 3, so this IS linear.
